## Install modules:
```npm install```

## Populate filesystem database:
```npm run populate```

## Run api server:
```npm run dev-api-server```

## Run dev server with vue-cli-service:
```npm run serve```

## Run store tests with mocha & chai:
```npm test```

## Run end-to-end tests with puppeteer:
```npm run e2e-test```
